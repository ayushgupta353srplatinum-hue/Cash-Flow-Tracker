# Cash Flow Tracker

A simple and responsive web application to track salary, expenses, and remaining balance.
Built using HTML, CSS, and vanilla JavaScript with focus on core front-end fundamentals.

# Features
- Add and save salary
- Add and delete expenses
- Automatic balance calculation
- Data persistence using LocalStorage
- Expense visualization using Pie Chart
- Mobile responsive design

# Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)
- Chart.js

# Project Structure
index.html
style.css
script.js

# How to Run
Open `index.html` in any modern web browser.

# Notes
This project is developed without any front-end framework to strengthen JavaScript fundamentals.
